Data Analyst Intern @ Nexon America

Summer 2024 Los Angeles, CA

During my 12-week internship at Nexon America, I worked on developing a Lifetime Value analysis system for MapleStory players, creating data visualization dashboards that would become integral tools for the marketing and game development teams.

Project Overview

The main focus of my internship was developing a system to track and analyze player lifetime value across different marketing campaigns. This involved working extensively with Snowflake and SQL for data analysis, and Tableau for visualization, ultimately creating tools that would help teams make data-driven decisions about marketing strategies and player engagement.

Technical Implementation

The project was implemented in several phases:

  • Database exploration and documentation using Snowflake SQL
  • Development of cohort analysis systems for marketing campaigns
  • Collaboration with Data Science team on player classification models
  • Creation of 3 comprehensive Tableau dashboards

Key Achievements & Impact

  • My project and it's findings were published in the company's Data Story monthly newsletter
  • My project was presented company-wide, receiving positive feedback from marketing and development teams
  • The project is currently being used to inform marketing strategies and game development
